About


In-depth study of user interfaces, user experience, data visualizations and scientific information presentation. This class covers aspects such as usability, psychology of the viewer, data science, some artistic theories, methods to best display information, and software tools that help accomplish better visuals. Students participate in a group project on UI/UX/data visualizations using modern methods. This is a project-based class, however, there will be student presentations, practical UI exercises, two exams, in-class activities and multiple homeworks.
